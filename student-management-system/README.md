#  Student Management System

 Project Overview
The Student Management System is a full-stack web application developed as part of a technical assessment. It allows users to manage student records efficiently, including adding, updating, viewing, and deleting student data.

Each student is assigned a unique Admission Number, which is automatically generated by the system.

---

##  Technology Stack

- Frontend: Angular  
- Backend: Node.js (Express.js)  
- Database: PostgreSQL  

---

##  Features

### Student Management
- Add new student
- Edit / update student details
- Delete student record
- View all students in a structured format

### Photo Upload
- Upload and store student photo
- Image saved on server (uploads folder)

### Unique Admission Number
- Automatically generated
- Ensures uniqueness for every student

##  Setup Instructions

### 1. Clone Repository

---

##  Database Setup

CREATE DATABASE student_db;

Run database.sql file

---

##  Backend Setup

cd backend  
npm install  
node app.js  

Server: http://localhost:3000

---

##  Frontend Setup

cd frontend  
npm install  
ng serve  

Frontend: http://localhost:4200
